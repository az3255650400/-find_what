# 🔬 生物分类查询助手

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Deployed-brightgreen)](https://yourusername.github.io/biological-classifier)
![License](https://img.shields.io/badge/License-MIT-blue.svg)

一个帮助学习生物分类学的Web应用，支持本地查询和AI增强模式。

## 🌟 功能特点

- 📚 查询常见生物的完整分类信息（界门纲目科属种）
- 🤖 AI模式支持查询任意生物（需API Key）
- 📱 响应式设计，适配手机和电脑
- 🚀 本地缓存，快速查询
- 🔒 API Key本地存储，隐私安全

## 🛠️ 技术栈

- HTML5 / CSS3 / JavaScript (纯前端)
- 通义千问API (DashScope) 或 Moonshot AI
- 本地存储 (localStorage)

## 🚀 快速开始
- 下载代码后 点击index.html即可
